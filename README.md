## 📖 О проекте

Данный проект представляет собой набор скриптов на **Python**, разработанных для автоматизации гидрологического моделирования в среде **GRASS GIS**. Основная задача — расчет энергетического баланса поверхности ледника через анализ солнечной радиации.

Скрипт интегрируется с модулем `r.sun` для вычисления ключевых метрик по каждой ячейке растра:
- **Приходящая солнечная радиация** (Global radiation)
- **Отраженная радиация** (Reflected radiation / Albedo effect)
- **Прямая и рассеянная составляющие**

Результаты расчетов (`Sin_cell` и производные) используются как входные данные для моделирования процессов таяния (абляции).

> ⚠️ **Статус:** Проект пока не завершен и находится в стадии **заброшенного (abandoned)**. Код может требовать адаптации под новые версии GRASS GIS или Python. Выкладывается в архивных целях.

## 🛠️ Технический стек

**Язык и окружение:**
- Python 3.x
- GRASS GIS 7.x (или 8.x)

**Ключевые модули GRASS:**
- `r.sun` — расчет инсоляции и радиационного баланса
- `r.mapcalc` — картографическая алгебра для постобработки
- `g.region` — управление пространственным_extent_ом

**Библиотеки Python:**
- `grass.script` / `grass.pygrass` — API для управления GRASS из Python
- `numpy` / `pandas` — обработка массивов данных
- `matplotlib` — визуализация результатов *(опционально)*

## 🚀 Начало работы

### Предварительные требования

- Установленный **GRASS GIS** (версия 7.8+ рекомендуется)
- Python 3.6+ с доступом к библиотекам GRASS
- Цифровая модель рельефа (DEM) исследуемого ледника
- Растр альбедо (коэффициента отражения) поверхности

### Установка и настройка

1. Клонируем репозиторий:
   
```bash
git clone https://github.com/Ratrio101/GlacierMeltModeling.git
cd GlacierMeltModeling
```

2. Запустите GRASS GIS и создайте Location с проекцией, соответствующей вашим данным
3. Импортируйте входные данные (DEM, альбедо) в mapset GRASS.
4. Настройте подходящие параметры в файле glacier_model.py.
5. Установите зависимости:
   
```bash
pip install requirements.txt
```

6. Запустите скрипт внутри сессии GRASS Python или через консоль.
```python
python glacier_model.py
```
